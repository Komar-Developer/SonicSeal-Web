A cryptographic chain of trust between file, software and license. If any link fails → silence.

How it works

  • 32-bit forensic watermark embedded in 15–18 kHz band — inaudible.
  • AES-256-GCM payload, key derived via HKDF(license, nonce).
  • Player authenticates via mutual HMAC handshake before decrypt.
  • Survives MP3 re-encoding, EQ, pitch-shift, stereo→mono, time-stretch.
  • 100% offline. No telemetry. No cloud uploads. Your audio never leaves your Mac.
